# AdventJS Challenges

Welcome to my AdventJS solutions! AdventJS is an advent calendar with programming challenges created by [@midudev](https://twitter.com/midudev).

Each year presents 25 challenges, one for each day leading up to Christmas. These exercises help improve JavaScript problem-solving skills while having fun.
